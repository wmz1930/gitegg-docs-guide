&emsp;&emsp;后台 Java 代码这里介绍使用 IntelliJ IDEA 进行开发，当然使用 Eclipse 也是可以的，这个可以根据自己的需要自由选择开发工具。

<a name="20bc145f"></a>

### 1. 下载需要的安装包，[IntelliJ IDEA 下载](https://www.jetbrains.com/idea/)，双击安装，一直点击下一步，如果 Windows 系统且有多个磁盘，尽量修改一下安装目录，不要安装在系统盘 C 盘即可。
